### Estructura Excel Fase 2 (recordatorio)

**Hoja CUADRO** (archivo de proyecto):
```
PISO | DPTO | TIPO | FICHA | COD | MUEBLE | CANTIDAD | CONJUNTO | SUB CONJUNTO
```
- FICHA alfanumérico (ej: "C02") → tiene archivo de ficha → corte
- FICHA numérico (ej: "100000121") → quincallería → stock directo

**Hoja B.DATOS** (cada ficha individual .xlsm):
```
Col E: N°FICHA | Col I: COD MATERIAL (→TABLERO) | Col R: LARGO | Col S: ANCHO
Col T: CANT UNI | Col U: VETA | Col V: PIEZA/INSUMO | Col W: COD PROG
Col Y: RAN | Col Z: MEC | Col AA: PER | Col AK: # TAPACANTO CUT RITE (→ENCHAPE)
```

**JOIN KEY:** prefijo del nombre del archivo = código de ficha  
`"C02_VERTICAL MODULAR.xlsm"` → ficha code = `"C02"`

**Cantidad:** `CUADRO.CANTIDAD × B.DATOS.CANT_UNI` (deduplicado por COD PROG)

## Comandos para arrancar en casa

```bash
# 1. Clonar o actualizar el repo
git clone https://github.com/Enriqueaad/Fabricacion-App.git
# o si ya lo tienes:
git pull origin master

# 2. Instalar dependencias
npm install

# 3. Crear la DB local (no está en el repo — está en .gitignore)
npx prisma migrate deploy
# o si es primera vez:
npx prisma migrate dev

# 4. Levantar el servidor
npm run dev
# → http://localhost:3002
```

> ⚠️ La carpeta `Base de Datos - APP/` con los Excel no está en el repo (archivos binarios grandes).
> Está en el ZIP: `Fabricacion-APP-extras.zip` en tu Desktop del trabajo.
> Cópialo a la misma ruta en casa: `C:\Users\[tu-user]\Documents\Desarrollos - APP\Fabricacion - APP\Base de Datos - APP\`

## Notas técnicas importantes

- **Next.js 16** — revisar `node_modules/next/dist/docs/` antes de cambiar config
- **Tailwind v4** — NO hay `tailwind.config.ts`, los tokens van en `@theme` dentro de `globals.css`
- **shadcn/ui v4** — usa OKLCH, no HSL. Init: `npx shadcn@latest init -b radix -p nova --css-variables`
- **XLSX server-side only** — no importar en client components
- **`generarExportCNC`** — inlineada en el wizard (evita bundlear xlsx en cliente)
- **maxDuration = 300** — necesario en la API route por el Excel de ~60MB
